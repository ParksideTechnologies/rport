Parkside Technologies RPort Home Assistant Add-on

This repository contains the RPort client add-on for Home Assistant.

To install it in Home Assistant:

1. Open Settings > Add-ons > Add-on Store.
2. Open the menu in the top-right corner and choose Repositories.
3. Add this repository URL:
   https://github.com/ParksideTechnologies/rport
4. Install the rPort add-on.
5. Enter the client username, password, client name, and optional UUID in the add-on configuration.

The add-on connects clients to the Parkside Technologies RPort server.
